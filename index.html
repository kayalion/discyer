Where do I find my personal access token?

Go to the developers page on Discogs. This page contains a button to (re)generate a personal access token.

The code after 'Current token:' is your personal access token and can be used to sign in.

About DiSCYeR

What about this website?

This website contains a web application to play music non-stop from your collection on Discogs.

Sign in, select one of your folders and a video from a random release will be played. You can view the details and when it's done playing the video, a next random release is selected and played and so forth... It just keeps on going.

You listen to your own record collection without worrying about flipping or switching the records.

Terms of service

This service comes as is and without guarantees. It seems to work though. The source code is also open-source and can be found on Github.

Privacy policy

This application does not store any personal data. The access token is kept client-side in memory and is only used to read from the Discogs API.
